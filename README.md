# Dashboard Contabil - Supermercado

App web para analisar planilhas de acumuladores fiscais.

## Como usar

1. Acesse o site
2. Arraste o arquivo `.xls` ou `.xlsx` (Empresa_XXXX_-_SUPERMERCADO)
3. O dashboard é gerado automaticamente

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy no Vercel

```bash
npm install -g vercel
vercel
```
